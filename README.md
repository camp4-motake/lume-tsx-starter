# Lume/TSX starter

Lume/TSX Static Site Development Environment

## Task

Please set up [Deno](https://deno.com/) v2.1 or higher first. `npm install -g deno` or [other installation](https://docs.deno.com/runtime/getting_started/installation/).

```sh
# dev
deno task serve

# build
deno task build

# zip
deno task zip
```
